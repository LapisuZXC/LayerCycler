LayerCycler — Плагин для Krita

LayerCycler помогает быстро переключаться между слоями и временно скрывать лишние слои. Плагин рассчитан на пользователей, которые не хотят копаться в настройках — всё максимально просто.

Функционал

  • Hide All but Top — скрывает все слои, оставляя видимым только верхний.
  • Hide All but Bottom — скрывает все слои, оставляя видимым только нижний.
  • Move Layer Up — переключает видимый слой на следующий сверху.
  • Move Layer Down — переключает видимый слой на слой ниже.
  • Вверху панели отображается название текущего видимого слоя.

Напоминание

Чтобы пользоваться плагином, не забудьте включить докер:

  • English: Settings → Dockers → Layer Cycler
  • Русский: Настройки → Докеры → Layer Cycler
